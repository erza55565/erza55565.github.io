Airlines online booking

Implementation of a couple microservices for the airlines booking system. Booking tickets, processing flights, routing, prices, discounts, etc. Also, the project had the internal layers of system services for big data processing such as recalculation prices, flight shifting, optimization, etc.

Team: 3 backend, 2 frontend, 1 devops.
Language: Java / Scala
Technologies: Docker, Spark, RabbitMQ, Cassandra
Platform: Cross-platform
Role: Architect / Core developer

  • Architecture.
  • BackEnd core features development.
  • Technical consultations.
  • Troubleshooting.
  • Interviewing new employers.
  • Solving difficult technical issues
The customer has improved speed of transaction processing (booking, searching, etc) Optimization of logistics and fees has been implemented.
